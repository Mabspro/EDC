# Bilingual Site Features for English & French Clients

## Recommended Site Elements

### 1. Language Toggle/Switcher
- Add a language selector in the navigation bar
- Allow users to switch between English (EN) and French (FR)
- Store language preference in localStorage
- Update content dynamically or show/hide language-specific sections

### 2. Bilingual Service Descriptions
- Display services in both languages
- Use tabs or accordion to show English/French versions
- Or show both languages side-by-side for key services

### 3. Contact Form Language Options
- Add language preference field to contact form
- Form labels and placeholders in both languages
- Thank you messages in selected language

### 4. Service Area Indicators
- Clear statement about serving English and French-speaking clients
- Geographic service areas mentioned
- Bilingual service capability highlighted

### 5. Key Information Bilingual
- Hero section subtitle mentioning bilingual services
- About section noting language capabilities
- Footer with bilingual contact information

### 6. Language-Specific CTAs
- Call-to-action buttons in both languages
- Service descriptions toggleable or shown together

## Implementation Approach

Since automatic translation by geography is not being used, we can:
- Add a manual language toggle
- Show key content in both languages
- Use expandable sections for detailed bilingual content
- Add language badges/indicators
